# Retail Sales & Customer Insights Dashboard

## Project Description

This project analyzes retail transaction data to uncover key business insights related to sales performance, customer behavior, and product profitability. The analysis involves data cleaning, exploratory data analysis (EDA), and SQL-based business queries to identify revenue drivers and purchasing patterns.

An interactive **Power BI dashboard** was developed to track important KPIs such as revenue, profit margin, average order value (AOV), and regional sales performance. The analysis revealed that the **top 20% of customers contributed nearly 68% of total revenue** and identified product categories with negative profit margins. 

The project demonstrates practical data analytics skills including **data preprocessing with Python (Pandas, NumPy), SQL-based analysis, and business intelligence visualization using Power BI** to support data-driven decision making.
